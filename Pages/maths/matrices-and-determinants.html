Matrices & Determinants

Array operations and solving simultaneous equations.

1. Matrix Operations

  • Addition/Subtraction: Matrices MUST have the exact same dimensions to be added or subtracted. Add corresponding elements.
  • Scalar Multiplication: Multiply every single element in the matrix by the scalar number.
  • Matrix Multiplication: The number of columns in the first matrix must equal the number of rows in the second matrix. (Row by Column multiplication).

2. Determinant of a 2x2 Matrix

For a matrix A containing elements: [a, b] top row, [c, d] bottom row.

|A| = ad - bc

A matrix with a determinant of zero is called a Singular Matrix. It has no inverse!

3. Inverse of a Matrix

The inverse A¯¹ multiplied by the original matrix A equals the Identity Matrix (I).

A¯¹ = (1 / |A|) × Adjugate(A)

For a 2x2 matrix, to find the adjugate: Swap the positions of a and d, and negate the signs of b and c.
